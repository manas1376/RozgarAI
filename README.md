RozgarAI is an AI-powered job search assistant designed to simplify and automate the job application process.

It helps users discover relevant job opportunities, analyze their resumes using ATS scoring, and track applications in a structured way.

🚀 Features:
- 🔍 Smart Job Search (LinkedIn, Naukri, etc. sources)
- 📊 ATS Resume Scoring (based on job description)
- 🧠 AI-based Job Matching
- 📁 Resume Upload & Analysis
- 📌 Application Tracker Dashboard
- 🔔 Notifications & Updates
- 🔐 Login & User Session (frontend prototype)

🛠️ Tech Stack:
- Frontend: React + Vite + Tailwind CSS
- UI: Modern responsive design

⚠️ Note:
This is currently a frontend-focused prototype. Backend integration, persistent authentication, and real-time data processing are planned in future updates.

🎯 Goal:
To build a fully automated AI-driven platform that can assist users in finding, analyzing, and applying for jobs efficiently.
